本项目名为“影片小本”——MovieNote，旨在给用户提供记录、评论、分享最近看过的电影的平台
目前项目包含 4 个页面，分别是登录页、个人主页、新记录提交页、搜索页

登录页面包含显示登录页面，点击“注册账号”之后打开注册面板，可在登录与注册之间切换。

个人主页包含个人影片记录展示，以及到个人信息、写新记录、访问他人主页的入口，但有些还没有实现
影片记录包含电影信息、观影记忆、个人照片三部分，其中包含两类图片：电影海报、用户当时拍下的照片

影片记录页面分上下两层，下层为记录编辑提交层，上层为照片编辑层——需要选择照片文件后点击编辑照片进入
编辑照片页面目前提供裁剪、滤镜、画笔三个重要功能

搜索页输入关键词可以搜索电影、用户、标签

网站已经完成路由映射、post解析、静态文件显示、模板渲染、数据库相关、保持登录状态等功能，
能够处理登录注册、展示个人主页、接收上传、搜索信息。
未实现的比较有用的需求有：用户编辑个人详细信息、用户关注其他用户、用户关注某个标签

本次作业说明：本作业新增三个功能：
	1、填写表单时会自动匹配数据库中的电影信息，如果该电影已经存在，就不需要输入电影信息，
	     只需要编辑影评和观影记忆部分
	2、新增标签：所有的电影信息展示中都包含其标签、用户可以为某个电影增加、删减标签
	3、搜索功能：在用户主页和搜索页都可以进行搜索、搜索结果分电影、用户、标签三类进行展示
	     	     其中标签会根据关联的电影的数量进行由多到少排序，还可以查看全部标签，也是热度由多到少
	4、个人主页右上角添加今日天气
	本次更新使项目的代码量由1800膨胀到2700，工作量很大有900行代码（不计算空行和依赖的npm包）
